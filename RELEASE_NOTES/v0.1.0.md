# v0.1.0 — Initial release

This initial release includes:

- Streamlit MVP app (`app.py`) for thesis input and report generation
- Heuristic scoring engine in `src/scoring/thesis_score.py`
- Market snapshot from `yfinance` in `src/data/market_data.py`
- Basic AI agent scaffolding in `src/ai/`
- GitHub Actions CI to run tests
- MIT License

Notes:
